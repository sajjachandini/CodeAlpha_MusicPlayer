# CodeAlpha_MusicPlayer
Student Information

Sajja Chandini is a 2nd Year B.Tech Artificial Intelligence and Data Science (AIDS) student interested in frontend development and web technologies.

Project Overview

The Music Player is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to play, pause, navigate between songs, control volume, and manage a playlist through an interactive and user-friendly interface.

Features

- Responsive Design
- Play and Pause Controls
- Next and Previous Song Navigation
- Song Title and Artist Display
- Progress Bar with Seek Functionality
- Current Time and Duration Display
- Volume Control
- Playlist with Song Selection
- Auto Play Next Song

Technologies Used

- HTML5
- CSS3
- JavaScript
